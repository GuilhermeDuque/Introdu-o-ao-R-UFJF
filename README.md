# Introdução-ao-R-UFJF
Aqui estarão os códigos e slides utiliizados no curso de introdução ao R para o curso de pós-graduação em Ciências Sociais na Universidade Federal de Juiz de Fora (PPGCSO-UFJF), entre os dias 08/12/2025 a 11/12/2025. Os códigos referente às aulas estão na pasta "códigos" e os slides na pasta "slides". Cada um dos códigos e apresentações serão postadas após a aula de referência.
